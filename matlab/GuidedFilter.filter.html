cv.GuidedFilter/filter

Apply Guided Filter to the filtering image

dst = obj.filter(src)
dst = obj.filter(src, 'OptionName',optionValue, ...)

Input

  • src filtering image with any numbers of channels.

Output

  • dst output image.

Options

  • DDepth optional depth of the output image. DDepth can be set to -1, which will be equivalent to class(src). Default -1
