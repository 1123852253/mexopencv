cv.HOGDescriptor/compute

Compute HOG descriptors

descs = hog.compute(im, 'Option', optionValue, ...)

Input

  • im Matrix of the type uint8 containing an image where objects are detected.

Output

  • descs Row vectors of hog descriptors, reshaped into a matrix with hog.getDescriptorSize() columns

Options

  • WinStride 2-element array [x,y]. defaults to CellSize
  • Padding 2-element array [x,y]
  • Locations cell array of 2-element arrays {[x,y],...} at which descriptors are computed.
