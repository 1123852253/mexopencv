MATLAB File Help: cv.DisparityWLSFilter/LRCthresh Index
cv.DisparityWLSFilter/LRCthresh

LRCthresh is a threshold of disparity difference used in left-right-consistency check during confidence map computation.

The default value of 24 (1.5 pixels) is virtually always good enough.
