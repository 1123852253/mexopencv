cv.BOWImgDescriptorExtractor/compute

Computes an image descriptor using the set visual vocabulary

[bow,idx] = extractor.compute(im, keypoints)

Input

  • im Image, for which the descriptor is computed.
  • keypoints Keypoints detected in the input image. It is a struct array that is returned by cv.FeatureDetector.

Output

  • bow Computed output image descriptor.
  • idx Indices of keypoints that belong to the cluster. This means that idx(i) are keypoint indices that belong to the i-th cluster (word of vocabulary).
