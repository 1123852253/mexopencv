cv.SuperpixelSEEDS/getNumberOfSuperpixels

Calculates the superpixel segmentation on a given image stored in object

num = obj.getNumberOfSuperpixels()

Output

  • num actual amount of superpixels.

The function computes the superpixels segmentation of an image with the parameters initialized with the constructor.
