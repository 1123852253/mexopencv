cv.EdgeAwareInterpolator/interpolate

Interpolate input sparse matches

dense_flow = obj.interpolate(from_image, from_points, to_image, to_points)

Input

  • from_image first of the two matched images, 8-bit single-channel or three-channel.
  • from_points points of the from_image for which there are correspondences in the to_image (a cell array of 2-element float vectors {[x,y],..}, size shouldn't exceed 32767).
  • to_image second of the two matched images, 8-bit single-channel or three-channel.
  • to_points points in the to_image corresponding to from_points (a cell array of 2-element float vectors {[x,y],..}, size shouldn't exceed 32767).

Output

  • dense_flow output dense matching (two-channel single image).
