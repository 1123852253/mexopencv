cv.SuperpixelLSC/getLabelContourMask

Returns the mask of the superpixel segmentation stored in object

img = obj.getLabelContourMask()
img = obj.getLabelContourMask('OptionName',optionValue, ...)

Output

  • img Return logical image mask where 1 indicates that the pixel is a superpixel border, and 0 otherwise.

Options

  • ThickLine If false, the border is only one pixel wide, otherwise all pixels at the border are masked. default true

The function return the boundaries of the superpixel segmentation.
