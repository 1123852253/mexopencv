MATLAB File Help: cv.EdgeAwareInterpolator/K Index
cv.EdgeAwareInterpolator/K

is a number of nearest-neighbor matches considered, when fitting a locally affine model.

Usually it should be around 128. However, lower values would make the interpolation noticeably faster.
