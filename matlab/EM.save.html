cv.EM/save

Saves the algorithm parameters to a file or a string

model.save(filename)
str = model.save(filename)

Input

  • filename Name of the file to save to. In case of string output, only the filename extension is used to determine the output format (XML or YAML).

Output

  • str optional output. If requested, the model is persisted to a string in memory instead of writing to disk.

This method stores the complete model state to the specified XML or YAML file (or to a string in memory, based on the number of output arguments).
