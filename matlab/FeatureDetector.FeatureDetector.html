cv.FeatureDetector/FeatureDetector

FeatureDetector constructors

detector = cv.FeatureDetector(type)

Input

  • type Type of the detector. see below. default 'FAST'

Output

  • detector New instance of the FeatureDetector

The following detector types are supported:

 'FAST'       FastFeatureDetector
 'STAR'       StarFeatureDetector
 'SIFT'       SiftFeatureDetector
 'SURF'       SurfFeatureDetector
 'ORB'        OrbFeatureDetector
 'MSER'       MserFeatureDetector
 'GFTT'       GoodFeaturesToTrackDetector
 'HARRIS'     GoodFeaturesToTrackDetector with Harris detector enabled
 'Dense'      DenseFeatureDetector
 'SimpleBlob' SimpleBlobDetector

Also a combined format with one of the following adaptor is supported

 'Grid'       GridAdaptedFeatureDetector
 'Pyramid'    PyramidAdaptedFeatureDetector

for example: 'GridFAST', 'PyramidSTAR'.
