cv.EM/trainE

Trains the GMM, starting from the Expectation step

[ log_likelihoods labels probs ] = em.trainE(samples, means0)
[ log_likelihoods labels probs ] = em.trainE(samples, means0, 'OptionName', optionValue, ...)

Input

  • samples Samples from which the Gaussian mixture model will be estimated. It should be a one-channel matrix, each row of which is a sample.

  • means0 Initial means ak of mixture components. It is a one-channel matrix of nclusters x dims size

Output

see EM.train

Options

  • Covs0 The vector of initial covariance matrices Sk of mixture components. Each of covariance matrices is a one-channel matrix of dims x dims size.

  • Weights0 Initial weights pi_k of mixture components. It should be a one-channel matrix with 1 x nclusters or nclusters x 1 size.
