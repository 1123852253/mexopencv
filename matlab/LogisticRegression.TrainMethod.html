cv.LogisticRegression/TrainMethod

Kind of training method used to train the classifier.

Default 'Batch'. Possible values:

  • Batch batch gradient descent.
  • MiniBatch Mini-Batch Gradient Descent. Set MiniBatchSize to a positive integer when using this method.
