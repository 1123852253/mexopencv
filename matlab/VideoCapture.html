cv.VideoCapture

VideoCapture wrapper class

Class for video capturing from video files or cameras. The class provides Matlab API for capturing video from cameras or for reading video files. Here is how the class can be used:

cap = cv.VideoCapture;
pause(3); % Note below
for t = 1:30
   imshow(cap.read);
   pause(0.1);
end

Note

In some environment, there is a concurrency issue during camera initialization. To avoid unexpected crash, pause for seconds after the initialization of VideoCapture object.
