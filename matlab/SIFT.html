MATLAB File Help: cv.SIFT Index
cv.SIFT

Class for extracting keypoints and computing descriptors using the Scale Invariant Feature Transform (SIFT).

Algorithm by D. Lowe [Lowe04].

References:

[Lowe04]:

David G Lowe. "Distinctive Image Features from Scale-Invariant Keypoints". International Journal of Computer Vision, 60(2):91-110, 2004.
