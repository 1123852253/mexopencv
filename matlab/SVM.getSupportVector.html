cv.SVM/getSupportVector

Retrieve a particular support vector.

sv = classifier.getSupportVector(index)

Input

  • index 0-based scalar index of support vector. The index must be between 0 and classifier.SupportVectorCount - 1.

Output

  • sv Support vector

The method can be used to retrieve a support vector by index.
