cv.DisparityWLSFilter.createRightMatcher

Convenience method to set up the matcher for computing the right-view disparity map that is required in case of filtering with confidence

matcher_right = cv.DisparityWLSFilter.createRightMatcher(matcher_left)

Input

  • matcher_left main stereo matcher instance that will be used with the filter. An object of one of the following classes:
    • cv.StereoBM stereo correspondence using the block matching algorithm.
    • cv.StereoSGBM stereo correspondence using the semi-global block matching algorithm.

Output

  • matcher_right output right matcher. An object of same class as matcher_left.
