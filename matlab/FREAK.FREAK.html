cv.FREAK/FREAK

Constructor

obj = cv.FREAK()
obj = cv.FREAK(..., 'OptionName',optionValue, ...)

Options

  • OrientationNormalized Enable orientation normalization. default true
  • ScaleNormalized Enable scale normalization. default true
  • PatternScale Scaling of the description pattern. default 22.0
  • NOctaves Number of octaves covered by the detected keypoints. default 4
  • SelectedPairs (Optional) user defined selected pairs indexes. Not set by default
