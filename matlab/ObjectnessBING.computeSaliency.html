MATLAB File Help: cv.ObjectnessBING/computeSaliency Index
cv.ObjectnessBING/computeSaliency

Compute the saliency

objectnessBoundingBox = obj.computeSaliency(img)

Input

  • img The input image, 8-bit.

Output

  • objectnessBoundingBox objectness Bounding Box vector. According to the result given by this specialized algorithm, the objectnessBoundingBox is a cell array of 4-element vectors. Each bounding box is represented by [minX, minY, maxX, maxY].

Performs all the operations and calls all internal functions necessary for the accomplishment of the Binarized normed gradients algorithm.
