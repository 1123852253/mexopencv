MATLAB File Help: cv.DisparityWLSFilter/SigmaColor Index
cv.DisparityWLSFilter/SigmaColor

SigmaColor is a parameter defining how sensitive the filtering process is to source image edges.

Large values can lead to disparity leakage through low-contrast edges. Small values can make the filter too sensitive to noise and textures in the source image. Typical values range from 0.8 to 2.0.
