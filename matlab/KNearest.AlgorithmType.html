MATLAB File Help: cv.KNearest/AlgorithmType Index
cv.KNearest/AlgorithmType

Algorithm type.

Default 'BruteForce'. Possible nearest neighbors implementations:

  • BruteForce brute-force search.
  • KDTree Fast Nearest Neighbor Search using KD-trees. This implements D. Lowe BBF (Best-Bin-First) algorithm for the last approximate (or accurate) nearest neighbor search in multi-dimensional spaces. See [BEIS97].
