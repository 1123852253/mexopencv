cv.minAreaRect

Finds a rotated rectangle of the minimum area enclosing the input 2D point set

rct = cv.minAreaRect(points)

Input

  • points Input 2D point set, stored in a cell array of 2-element vectors or 1-by-N-by-2 numeric array.

Output

  • rct Output rotated rectangle.

The function calculates and returns the minimum-area bounding rectangle (possibly rotated) for a specified point set.
