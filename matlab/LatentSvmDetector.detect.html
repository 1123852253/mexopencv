cv.LatentSvmDetector/detect

Detects objects

detections = detector.detect(im, 'Option', optionValue, ...)

Input

  • im Matrix of the type uint8 containing an image where objects are detected.

Output

  • detections Struct array of detected objects. It has the following fields:
    • rect rectangle [x,y,w,h] of the object
    • score score of the detection
    • class name of the object class

Options

  • OverlapThreshold Parameter to specify the threshold. default 0
  • NumThreads Number of parallel threads when OpenCV is built with TBB option. default -1
