cv.DisparityWLSFilter/getConfidenceMap

Get the confidence map that was used in the last filter call

map = obj.getConfidenceMap()

Output

  • map confidence map. It is a single one-channel image with values ranging from 0.0 (totally untrusted regions of the raw disparity map) to 255.0 (regions containing correct disparity values with a high degree of confidence).
