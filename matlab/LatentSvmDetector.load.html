cv.LatentSvmDetector/load

Loads a model from files

status = detector.load(filenames)
status = detector.load(filenames, classnames)

filenames is a string or a cell array of strings specifying the location of model files. Optionally the method takes class names as a cell array of strings that has the same size to filenames. By default, class names are set to the name of the file without xml extension. S is a logical value indicating success of load when true.
