cv.VideoCapture/get

Returns the specified VideoCapture property

Input

  • prop Property identifier. It can be specified as a string (one of the recognized properties), or directly as its corresponding integer code.

Output

  • value Value of the property (as a double).

When querying a property that is not supported by the backend used by the cv.VideoCapture class, value 0 is returned.

Example

All the following are equivalent:

b = cap.Brightness
b = cap.get('Brightness')
b = cap.get(10)  % enum value defined in OpenCV source code
