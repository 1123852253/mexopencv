cv.EM/trainM

Trains the GMM, starting from the Maximization step

[ log_likelihoods labels probs ] = em.trainM(samples, probs0)

Input

  • samples Samples from which the Gaussian mixture model will be estimated. It should be a one-channel matrix, each row of which is a sample.

  • probs0 Initial probabilities p_i;k of sample i to belong to mixture component k. It is a one-channel floating-point matrix of nsamples x nclusters size

Output

see EM.train
