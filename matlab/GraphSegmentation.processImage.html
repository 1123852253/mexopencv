cv.GraphSegmentation/processImage

Segment an image and store output in dst

dst = obj.processImage(src)

Input

  • src The input image. Any number of channel (1 (Eg: Gray), 3 (Eg: RGB), 4 (Eg: RGB-D)) can be provided.

Output

  • dst The output segmentation. It's a int32 matrix with the same number of cols and rows as input image, with an unique, sequential, id for each pixel.
