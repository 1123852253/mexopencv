cv.HausdorffDistanceExtractor/RankProportion

The rank proportion (a fractional value between 0 and 1) that establish the K-th ranked value of the partial Hausdorff distance.

Experimentally had been shown that 0.6 is a good value to compare shapes. default 0.6
