cv.MSER/detectRegions

Maximally stable extremal region extractor

Input

  • image Input 8-bit grayscale or color image (supports 1/3/4-channels).

Output

  • msers The output vector of connected points (list of point sets). Cell-array of cell-array of 2D points {{[x,y],[x,y],..}, {[x,y],..}}.
  • bboxes Output vector of rectangles (bounding boxes). A cell-array of 4-element vectors {[x,y,width,height], ...}.

Runs the extractor on the specified image; returns the MSER regions, each encoded as a contour (see cv.findContours).
