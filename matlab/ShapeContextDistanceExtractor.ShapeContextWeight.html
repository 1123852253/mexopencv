cv.ShapeContextDistanceExtractor/ShapeContextWeight

The weight of the shape context distance in the final value of the shape distance.

The shape context distance between two shapes is defined as the symmetric sum of shape context matching costs over best matching points. The final value of the shape distance is a user-defined linear combination of the shape context distance, an image appearance distance, and a bending energy. default 1.0
