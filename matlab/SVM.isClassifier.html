cv.SVM/isClassifier

Returns true if the model is a classifier

b = model.isClassifier()

Output

  • b Returns true if the model is a classifier ('C_SVC', 'NUSVC', or 'ONECLASS'), false if the model is a regressor ('EPSSVR', 'NUSVR').
