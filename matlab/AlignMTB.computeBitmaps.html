cv.AlignMTB/computeBitmaps

Computes median threshold and exclude bitmaps of given image

[tb,eb] = obj.computeBitmaps(img)

Input

  • img input image (uint8 grayscale).

Output

  • tb median threshold bitmap, of same size as img and uint8 type.
  • eb exclude bitmap, of same size as img and uint8 type.
