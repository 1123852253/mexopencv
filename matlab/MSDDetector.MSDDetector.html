cv.MSDDetector/MSDDetector

The full constructor

obj = cv.MSDDetector()
obj = cv.MSDDetector('OptionName',optionValue, ...)

Options

  • PatchRadius Patch radius. default 3
  • SearchAreaRadius Search Area radius. default 5
  • NMSRadius Non Maxima Suppression spatial radius. default 5
  • NMSScaleRadius Non Maxima Suppression scale radius. default 0
  • ThSaliency Saliency threshold. default 250.0
  • KNN Number of nearest neighbors. default 4
  • ScaleFactor Scale factor for building up the image pyramid. default 1.25
  • NScales Number of scales number of scales for building up the image pyramid (if set to -1, this number is automatically determined). default -1
  • ComputeOrientation Flag for associating a canoncial orientation to each keypoint. default false
