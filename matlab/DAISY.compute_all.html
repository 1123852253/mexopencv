cv.DAISY/compute_all

Compute all descriptors of an image

descriptors = obj.compute_all(image)
descriptors = obj.compute_all(image, roi)

Input

  • image image to extract descriptors.
  • roi optional region of interest within image [x,y,width,height].

Output

  • descriptors In the first variant, descriptors are computed for all image pixels. In the second variant, resulted descriptors array for ROI image pixels.
