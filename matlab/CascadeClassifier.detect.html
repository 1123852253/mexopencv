cv.CascadeClassifier/detect

Detects objects of different sizes in the input image.

boxes = classifier.detect(im, 'Option', optionValue, ...)

The detected objects are returned as a cell array of rectangles.

Input

  • im Matrix of the type uint8 containing an image where objects are detected.

Output

  • boxes Cell array of rectangles where each rectangle contains the detected object.

Options

  • ScaleFactor Parameter specifying how much the image size is reduced at each image scale.
  • MinNeighbors Parameter specifying how many neighbors each candiate rectangle should have to retain it.
  • MinSize Minimum possible object size. Objects smaller than that are ignored.
  • MaxSize Maximum possible object size. Objects larger than that are ignored.
